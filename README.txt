BEErich Hive Tycoon – instrukcja uruchomienia

1. Wypakuj ZIP.
2. Umieść wszystkie pliki na dowolnym serwerze WWW (np. GitHub Pages, Netlify).
3. Otwórz index.html w przeglądarce.
4. Klikaj “Zbieraj miód” → “Sprzedaj miód” → ulepszaj pszczoły i ule.
5. Co 30 sekund wyskakuje powiadomienie z linkami do Twoich sociali.
6. Masz pełne prawo do rozwijania gry.
